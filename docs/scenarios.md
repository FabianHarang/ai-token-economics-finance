# Scenarios

## Baseline Global Compute

EU users retain access to global compute. Token prices remain comparatively stable while foreign compute dependency stays high.

## EU-Only Compute Shock

From 2028, public-sector and regulated users must use EU-executed compute. EU-local demand jumps into a constrained supply base and token scarcity rents emerge.

## Fast Grid Buildout

Uses the same sovereignty shock but accelerates grid connection expansion. Grid scarcity rent falls and more demand is served.

## AI Efficiency Breakthrough

Improves tokens per MWh faster than the baseline. Electricity intensity falls, capacity stretches further, and rebound demand partly offsets the gain.

## Public-Sector Priority

Public services receive first access to constrained EU-local compute. Public unmet demand falls while firms and consumers absorb more rationing.

## US/EU Compute Alliance

EU users retain access to certified allied compute. Price and rationing shocks are muted, but strategic dependency remains.
