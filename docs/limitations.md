# Limitations

This is not a forecast. It is a scenario exploration model.

Results depend on assumptions, especially demand growth, local compute shares, data-centre build times, grid-connection delays, electricity-price elasticity, and AI-efficiency progress.

The browser version uses reduced-form electricity markets with a seeded stochastic weather path. It does not perform hourly dispatch, full power-flow modeling, carbon accounting, detailed procurement simulation, strategic oligopoly, contract-level data-centre economics, or Monte Carlo confidence intervals over many weather seeds.

Real data-centre contracts, GPU inventories, model-provider margins, and government compute procurement terms are partly proprietary. Defaults should be treated as transparent placeholders until replaced with source-linked calibration.
